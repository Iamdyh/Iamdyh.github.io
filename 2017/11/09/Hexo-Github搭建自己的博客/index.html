Hexo+Github搭建自己的博客

发表于 2017-11-09

今天记录一下用Hexo + Github搭建自己的博客：
使用平台：windows7 64位
1、安装Git：
在官网或者其他地方下载安装git,这里我使用的版本是：git-2.8.3-64-bit。
2、安装Node.js：
在官网或者其他地方下载安装git,这里我使用的版本是：node-v8.9.0-x64。
3、安装并搭建Hexo：
3、1：随便在电脑的某个盘中新建一个文件夹（比如我是在D盘中新建了一个名为blog的文件夹），此文件夹就是以后自己博客的根目录。
3、2：在此文件夹内使用git bash或者打开cmd,在cmd中访问到此目录下，然后输入如下两句命令：

 npm install -g cnpm --registry=https://registry.npm.taobao.org
 cnpm install hexo-cli -g

       注：由于国内npm源有问题，所以命令：npm install hexo -cli -g 无法成功安装hexo，使用上面两句命令后就可以安装成功了。
 3、3：Hexo插件，建议一起安装：

$ cnpm install hexo-generator-index --save #索引生成器
$ cnpm install hexo-generator-archive --save #归档生成器
$ cnpm install hexo-generator-category --save #分类生成器
$ cnpm install hexo-generator-tag --save #标签生成器
$ cnpm install hexo-server --save #本地服务
$ cnpm install hexo-deployer-git --save #hexo通过git发布（必装）
$ cnpm install hexo-renderer-marked-enhanced --save #渲染器
$ cnpm install hexo-renderer-stylus@0.3.0 --save #渲染器

    注：由于npm换源了，所以以后npm都变成了cnpm。
 3、4：初始化：
          在cmd中访问到刚才新建的blog文件夹目录，然后输入命令：
          hexo init
 3、5：生成静态页面：
          输入命令: 
          hexo g
 3、6：本地服务：
          先安装hexo server，否则没有用，输入命令：
          hexo s
  到此，本地Hexo搭建成功，测试网站：http://localhost:4000，如果能正常打开，说明成功，否则重复前面的步骤，再试一次。

4、配置GitHub：
4、1：首先如果没有github账号就先申请一个，然后新建一个repository,名字为：yourusename.github.io，yourusename为你自己的github用户名。
4、2：在上面新建的blog文件夹中找到 “_config”文件，然后编辑：
deploy:
type: git
repo: https://github.com/Iamdyh/Iamdyh.github.io.git
branch: master
注意：网址需要改成刚才自己新建仓库的地址，而且上面的git、https、master前面均需要有空格。
4、3：如果是第一次，则在cmd中执行如下命令：

cnpm install hexo-deployer-git --save
hexo d
如果不是，只需要执行
hexo d

5、博客搭建完成：
完成上述步骤后，访问：https:yourname.github.io,就可以看到自己的博客了。注:yourname为自己的github用户名。
